布局结构

  • 全局导航栏：保留网站原有头部（创作者工具箱 + 反馈/计费管理/头像）
  • 编辑器顶栏：← 返回 + 可编辑标题（聚焦蓝色边框）+ 状态Badge + 🗑清空 + ⬇导出 + 蓝色保存按钮
  • 左栏：素材面板（搜索/素材库 Tab），卡片可拖拽
  • 中栏：Tiptap 工具栏 + 编辑区 + 底部状态栏
  • 右栏：AI 功能按钮 + 任务进度（hover 显示删除按钮）
