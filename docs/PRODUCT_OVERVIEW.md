# JoyfulWords 产品简介（精简版）

## 1. 产品定位

JoyfulWords 是面向内容创作者的 AI 创作工作台，将素材搜索、文章创作、内容编辑和多形态输出集中在同一个工作流中，减少在多个工具之间复制、粘贴和重复制作。

## 2. 主要功能

| 功能 | 简介 |
| --- | --- |
| 素材搜索与素材库 | 搜索资讯、新闻和图片，将有用内容收藏并复用于创作。 |
| AI 文章创作 | 根据主题、素材和写作要求生成大纲与初稿。 |
| 文章编辑 | 支持富文本编辑、AI 改写、续写、翻译和内容优化。 |
| 视觉内容生成 | 生成文章封面、配图、信息图和 AI 图表。 |
| 多形态输出 | 将文章转换为 PPT、思维导图、播客音频、Word 或微信公众号格式。 |
| 任务中心 | 集中查看异步任务进度、结果、失败原因，并下载或重试。 |
| MCP 接入 | 可通过标准 HTTP MCP Server 连接 Claude Code 或其他 AI Agent。 |
| Free Tools | 提供独立的轻量创作与文档转换工具，部分工具可免登录使用。 |

## 3. 如何使用

1. 注册或登录账号，新用户可获得 200 积分。
2. 搜索并收藏素材，或直接输入自己的选题和写作要求。
3. 使用 AI 生成文章初稿，在编辑器中继续修改和完善。
4. 按需要生成配图、信息图、PPT、思维导图或播客音频。
5. 在任务中心查看进度，完成后下载、导出或继续编辑。

## 4. 产品价格

JoyfulWords 无订阅费，采用充值积分、按实际用量扣费的模式；**1 积分 = 0.01 美元**，最低充值 200 积分。

| 项目 | 价格 |
| --- | --- |
| 注册体验 | 赠送 200 积分 |
| 订阅费 | 0 |
| AI 文章 | 预计 20–80 积分/篇，按实际模型用量计算 |
| 普通文字模型 | 输入 1 积分/千 Token；输出 3 积分/千 Token |
| 高级文字模型 | 输入 2 积分/千 Token；输出 10 积分/千 Token |
| AI 生图 | 1–8 积分/张，按模型计算 |
| 素材搜索 | 资讯/新闻 1 积分/次；图片 2 积分/次 |
| PPT 生成 | 1 积分/页 |
| 免费能力 | 编辑器、素材库、思维导图、Word 和微信公众号格式转换 |

实际消耗会受内容长度、模型、图片数量和生成次数影响，以账户使用记录为准。

## 5. 适用人群

- **自媒体与公众号作者**：提高选题、写作、配图和排版效率。
- **博客与独立创作者**：从一个想法快速完成文章和多平台内容。
- **内容营销与出海团队**：制作中英文内容、视觉素材和演示文稿。
- **职场人士与顾问**：将报告、方案和文章整理为 PPT 或思维导图。
- **教师与知识工作者**：把长内容转为课程大纲、图表和演示材料。
- **开发者与 AI Agent 用户**：通过 MCP 在自动化工作流中调用 JoyfulWords。

## 6. 核心价值

- 一个工作台完成“找素材—写文章—做视觉—多格式导出”。
- 同一份内容可持续编辑、复用，并转换为多种交付形式。
- 无月费，按量付费，适合从轻度体验到高频创作。
